This project was originally meant to be a replica of the “ILOVEYOU” virus, but I ended up changing a lot of things. Now it’s more of a ransomware-style program.

It encrypts files and then interacts with the user in an annoying/cringy way. After doing certain things, it gives you a way to get your files back (like a code or instructions to decrypt them).

This is just for learning and experimenting — mainly to understand how encryption and basic ransomware concepts work in a controlled environment.
